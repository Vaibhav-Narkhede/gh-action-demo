# gh-action-demo

Vite + React + TypeScript starter app.

## Run

```bash
npm install
npm run dev
```

## Go hello world

Run the Go command locally:

```bash
go run ./cmd/hello
```

To publish Linux, macOS, and Windows binaries in a GitHub Release, push a version tag:

```bash
git tag v1.0.0
git push origin v1.0.0
```

The `Release Go binary` workflow also supports manual runs, which produce a downloadable workflow artifact without creating a GitHub Release.
